# ✈️ 1942 REBORN

> 태평양 상공을 누비며 32스테이지를 돌파하라! 붉은 돼지 풍 레트로 복엽기로 출격!

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/1942-reborn](https://mindweaver74.github.io/1942)

---

## 🎮 게임 방법

1. 화면을 **드래그**해서 전투기를 이동하세요
2. 총알은 **자동 발사**됩니다
3. 적기를 모두 격추하면 다음 스테이지로 진행
4. **4스테이지마다 보스** 등장
5. **32스테이지** 클리어가 최종 목표!

## ✈️ 비행기 디자인

**플레이어기 (사보이아 S.21 풍)**
- 붉은 유선형 동체
- 위아래 복엽 날개
- 회전하는 프로펠러
- 오픈 콕핏 + 고글 조종사

**적기 (레트로 복엽기 풍)**
- 소형 복엽기 — 4가지 타입
- 각 타입별 색상과 HP

**보스기 (대형 비행정 풍)**
- 양쪽 엔진 2개 + 이중 프로펠러
- 5방향 탄막 공격
- HP바 표시
